# Tortoise-and-Hare-race-simulator
This is a simple Python simulation of the classic Tortoise and Hare fable, 
using the random library to determine random positions for Hare and Tortoise.

The tortoise is slow but steady.
The hare is fast but unpredictable.
Who wins? Run it and find out.

## Features
- Randomized position at each time step
- Clear step-by-step text race visualization
- Replayable with a new outcome
